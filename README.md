# AI PM OS Dashboard V5

面向 AI 产品经理的智能操作系统 - 学习 · 拆解 · 训练 · 成长

## ✨ 功能特性

### 🎯 六大核心模块

1. **今日任务系统**
   - 添加、完成、删除任务
   - 实时统计（总任务/已完成）
   - 数据自动保存到 LocalStorage

2. **产品拆解中心**
   - 9维产品拆解框架
   - ChatGPT 拆解示例
   - 自定义产品拆解记录

3. **Prompt 训练中心**
   - RCGCF 公式（Role, Context, Goal, Constraint, Format）
   - 好/差 Prompt 对比示例
   - Prompt 收藏库（支持分类）

4. **产品经理成长地图**
   - 5个成长阶段可视化
   - 进度追踪
   - 阶段说明：
     1. AI 使用者
     2. Prompt 工程师 ← 当前阶段
     3. AI 产品经理
     4. AI Builder
     5. AI Founder

5. **项目管理系统**
   - 项目添加与状态管理
   - 三种状态：规划中 / 进行中 / 已完成
   - 项目列表展示

6. **今日复盘**
   - 每日学习总结
   - 历史复盘记录查看
   - 自动日期标记

## 🎨 设计特点

- **深色主题**：护眼且专业
- **玻璃拟态效果**：现代感十足
- **精美动画**：流畅的交互体验
- **响应式设计**：适配各种屏幕尺寸
- **独特字体**：Playfair Display + DM Sans

## 💾 数据持久化

所有数据使用 LocalStorage 自动保存：
- 任务数据 (`pm_tasks`)
- 项目数据 (`pm_projects`)
- 复盘记录 (`pm_reviews`)
- Prompt 收藏库 (`pm_prompts`)
- 产品拆解 (`pm_decomposition`)

## 🚀 快速开始

1. 克隆仓库或下载 `AI_PM_OS_Dashboard_V5.html`
2. 用浏览器打开文件即可使用
3. 无需安装任何依赖，纯前端实现

- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript (ES6+)
- LocalStorage API
- Google Fonts (Playfair Display, DM Sans)
